(1) In your git directory, git clone wtwong316/frappe_docker, wtwong316/erpnext and wtwong316/frappe 
    $git clone https://github.com/wtwong316/frappe_docker.git
    $git clone https://github.com/wtwong316/erpnext.git 
    $git clone https://github.com/wtwong316/frappe.git

(2) In your frappe_docker git directory, just perform the local setup now, copy the env-local to .env
    $cp env-local .env

(3) If need to stop all running containers first
    $ docker container stop $(docker container list -q)

(4) In your frappe_docker git directory, perform docker compose command
    $docker-compose --project-name oxytrack up -d
